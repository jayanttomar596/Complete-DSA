# DSA Mastery: A Complete Data Structures & Algorithms Learning Journey

A comprehensive collection of **100+ DSA problems and implementations** covering all major topics. This repository documents my complete learning journey from basics to advanced algorithms, with clean, well-commented solutions in C++.

## Topics Covered

### Fundamentals
- **Basic Concepts** - Complements, Conversions, Pattern Questions, Reversals
- **Pointers & Memory** - Pointer arithmetic, double pointers, function pointers, dynamic memory allocation
- **References** - Reference variables and their usage
- **Keywords & Modifiers** - Default arguments, inline functions, macros

### Core Data Structures
- **Arrays** - Searching, Sorting, Intersection, Pair Sum, Unique Elements
- **Strings** - Palindrome, Character Frequency, Manipulation
- **Linked Lists** - Single, Doubly, Circular with comprehensive operations
- **Stacks & Queues** - Implementation, Applications, Circular Queues
- **Hash Maps** - Hash tables, Problem-solving with hashmaps

### Advanced Data Structures
- **Heaps** - Min/Max heaps, Heap Sort, Priority Queues
- **Binary Trees** - Traversals, BST operations, Tree problems
- **Tries** - Prefix trees, autocomplete implementations
- **Graphs** - Creation, Traversals (BFS/DFS), Algorithms

### Algorithms
- **Searching** - Linear, Binary Search variants
- **Sorting** - Bubble, Selection, Merge, Quick, Heap Sort (with comparisons)
- **Recursion** - Base cases, Tree/String problems, Optimization
- **Backtracking** - N-Queens, Rat Maze, Sudoku Solver
- **Dynamic Programming** - Memoization, Tabulation, 2D DP problems
- **Greedy Algorithms** - Activity Selection, Fractional Knapsack, Huffman Coding
- **Graph Algorithms** - Shortest paths, Minimum Spanning Trees

### Specialized Topics
- **Matrix Multiplication** - Strassen's Algorithm
- **Mathematical Concepts** - GCD, Prime Numbers, Modular Exponentiation
- **STL Deep Dive** - Vectors, Maps, Sets, Priority Queues, Algorithms

## Getting Started

### Prerequisites
- **C++ Compiler**: GCC, Clang, or MSVC
- **CMake** (optional, for building)
- **IDE**: VS Code, CLion, or any C++ IDE

### Compiling & Running

**Single file:**
```bash
clang++ -std=c++17 filename.cpp -o output
./output
```

**With GCC:**
```bash
g++ -std=c++17 filename.cpp -o output
./output
```
