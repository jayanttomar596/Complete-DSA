# Quick Start Guide

## 🚀 Getting Started

### For Complete Beginners
1. Start with `BASIC/` folder
2. Read through Pointers, Dynamic Memory, and References
3. Move to Arrays and Strings in `DSA/ARRAYS/`
4. Then learn Data Structures (Linked Lists, Stacks, Queues)

1. Review **Array problems** - [DSA/ARRAYS/](DSA/ARRAYS/)

## 💻 How to Compile and Run

```bash
# Navigate to the file location
cd DSA/ARRAYS/

# Compile
g++ -std=c++17 reverse_array.cpp -o reverse_array

# Run
./reverse_array

# Or use clang
clang++ -std=c++17 reverse_array.cpp -o reverse_array
./reverse_array
```

## 📚 Complexity Cheat Sheet

| Data Structure | Insert | Delete | Search | Space |
|---|---|---|---|---|
| Array | O(n) | O(n) | O(n) | O(n) |
| Linked List | O(1) | O(1) | O(n) | O(n) |
| Stack | O(1) | O(1) | O(n) | O(n) |
| Queue | O(1) | O(1) | O(n) | O(n) |
| BST | O(log n) | O(log n) | O(log n) | O(n) |
| Hash Table | O(1) | O(1) | O(1) | O(n) |
| Heap | O(log n) | O(log n) | O(n) | O(n) |
| Trie | - | - | O(m) | O(n) |

## 🎓 Tips for Maximum Learning

1. **Understand, Don't Memorize** - Know WHY, not just HOW
2. **Trace Through** - Walk through examples step by step
3. **Modify and Experiment** - Change code and see what happens
4. **Compare Approaches** - See multiple solutions
5. **Analyze Complexity** - Always know the time/space complexity
6. **Practice Consistently** - Daily practice > weekly cramming
7. **Teach Others** - Explain concepts to solidify learning

## ❓ FAQs

**Q: Where should I start?**  
A: Start with BASIC/ folder if new to C++, then move to DSA fundamentals.

**Q: How long does it take to master DSA?**  
A: 2-3 months with consistent daily practice (2-3 hours).

**Q: Should I memorize all solutions?**  
A: No! Understand patterns, not individual solutions.

**Q: How do I prepare for interviews?**  
A: Review common patterns, solve problems under time pressure, explain your approach.

**Q: Which topics are most important?**  
A: Arrays, Strings, Linked Lists, Trees, and Dynamic Programming.
